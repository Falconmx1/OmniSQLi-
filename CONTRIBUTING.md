# 🤝 Guía de Contribución - OmniSQLi

¡Gracias por querer contribuir a OmniSQLi! Esto va a estar cabrón.

## 📋 ¿Cómo contribuir?

### 1. Reportar Bugs
- Usa el issue tracker
- Describe paso a paso cómo reproducirlo
- Incluye logs, capturas, payloads usados
- Menciona tu SO y versión de Python

### 2. Sugerir Features
- Describe la funcionalidad
- Explica por qué sería útil
- Si puedes, da ejemplos de uso

### 3. Pull Requests
- Fork el repo
- Crea una rama: `git checkout -b feature/MiFeature`
- Haz commit: `git commit -m 'Añadí tal cosa'`
- Push: `git push origin feature/MiFeature`
- Abre el Pull Request

- Usa Python 3.8+
- Sigue PEP 8
- Comenta el código en español o inglés
- Añade docstrings a funciones importantes
